# Entorno virtuales python

## Crear un entorno virtual

```
python -m venv venv
```

## Activar el entorno virtual

```
venv\Scripts\activate
```

## Desactivar el entorno virtual

```
deactivate
```

# Flask


# Verificar que flask se haya instalado
